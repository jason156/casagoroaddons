=================
Location Security
=================

A los usuarios se les puede asignar muchos Diarios de Bolsa y luego se les puede restringir para ver solo estos Diarios.

Instalación
============

Para instalar este módulo, necesita:

#. Solo es necesario instalar el módulo.

Configuración
=============

Para configurar este módulo, necesita:

#. Vaya a Configuración / Usuarios / Usuario, seleccione un usuario y luego en Preferencias establezca "Restringir ubicación".
#. En la página "Ubicaciones de stock permitidas" establezca las ubicaciones que permite.
