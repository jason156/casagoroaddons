========================
Website Sale Wishlist UX
========================

This module let to select the variant you want to add the the Wishlist instead of
adding the default one. It works the same as the shopping cart does from the products page:

* if the product has only one variant will automatically add the product to the Wishlist
* if the product has more variants, will leave you to the product page so there you can select the proper attributes and then add it to the Wishlist
