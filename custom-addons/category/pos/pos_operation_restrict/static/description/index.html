Odoo 13 POS: User to set discount and price

POS Operation Restriction module allow user to set discount and price if they have rights for change it else Manager's authentication required.

Note: This module is tested and working in Odoo community edition but not tested in Enterprise version. And we are supporting Ubuntu OS and not Windows OS.

Available for Enterprise

How to use

  • First of all, we have to set the configuration in Settings > Users .

  • We have to set the barcode of user .

  • Also need to set pin of user in .

  • Define users as a manager in POS configuration.

  • When discount value exceed then user's discount limit a popup open for manager authentication.
  • In the popup scan or type a barcode of any manager.

  • If the manager has rights to give discount then discount will be applied in product.
