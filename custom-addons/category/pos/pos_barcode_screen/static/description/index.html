POS Barcode Screen

Provides better screen for scanning products using barcode in POS. Also, seller will able to see the detailed view of scanned product along with product image.

Generally in POS, whenever a seller scans a product from barcode reader, product is added in the cart which are unidentifiable due to the small size of cart. So, implementing Odoo POS Barcode Screen‚Äù which allows you to display the different screen with the detailed view of product whenever product is scanned.

Features:

  • A new screen appears when product is scanned using barcode
  • Detailed view of products are shown in barcode screen.
  • A bigger image with product description for customer is shown for selected orderline.
  • Seller can activate/deactive barcode screen view at any time in running session.
