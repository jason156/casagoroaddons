POS Test/Sample Orders

This application will help you to test POS Orders and its payment. Testing order will not affect actual order sequence and there will not be any extra entry generated for that order in the system.

Key features:
  • User can test order by creating order with Test Option enable in Payment screen of POS in live server.
  • Actual order sequence will not be effect. i.e. If Test order generated with order number 00001-001-0001, than after new regular order generated by user, than system will show the same order number 00001-001-0001 for that regular order.
  • Test/Sample orders will not be saved(database) as like regular order in backend.

After installing POS Test Order module, to enable the Test Order feature, You need to set Test Orders option in POS Configuration. Inside POS Configuration, you can find options as follows:

  • Test Orders - enable/disable Test button
