How to Setup

1. HOW IT WORKS

"Queue Post Order" switches odoo's default routing point of sale to the queue table before posting it to the stock and journal entries. This is necessary when there are many Point of sale transactions that are blocked from entering due to resource limitations of Postgresql servers or internet connections.

The queue list can be seen on the "Pos Order Queue" menu.

You can set the interval in the Post Order Queue on the "Scheduled Actions" menu, then enter the parameters in the form of an integer that matches your server's resources.
