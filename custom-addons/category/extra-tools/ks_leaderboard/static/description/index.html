Overview

Ksolves Leaderboard module provides quick visual reference of your customer related entities like Sales Order, Purchase Orders, Contacts, Bank Accounts, Employee details etc . The module simply turns your data into insights, giving the option to compare different entities data and hence instinctive understanding of your customers and business operations!

PROBLEM:

With Odoo default modules, it is not possible to refactor your module data and display it graphically on a Dashboard.

SOLUTION:

Ksolves Leaderboard module offers a clear visual representation of your valuable module data in form of robust cards on a Leaderboard. It lets you compare the data based on different filters and layout options

Key Features

  • Enables quick and clear visual representation for almost all Odoo entities.

  • Create/Configure Leaderboard
  • You can configure your Leaderboards from the menu My Leaderboard > Configuration > Leaderboards menu as shown below:
  • On clicking the Leaderboard menu, you will be able to see the below screen:
  • Here you can create Leaderboard or modify the existing Leaderboard as per your need
  • While creating/modifying Leaderboard :
  • You can activate and deactivate a customized Leaderboard by checking and unchecking the box respectively
  • Group Access option allows you to set groups permission that who can access the selected Leaderboard.
  • Add a Leaderboard Item:
  • Click on the Add Item Button on the top right corner of your Leaderboard screen.
  • Once you click on the Add Item Button you will notice the following item form page view.
  • Fields which are highlighted in purple are mandatory to fill, model field is the important one in order to save the Leaderboard item.
  • While filling the fields you will notice the preview on the right side of the screen and it will keep reflecting the changes based on the data that you have filled.
  • After filling the data section of item, you can choose various Header and Body layout with various themes and gradients.
  • After filling all required fields, you can save the item and go back to previous Leaderboard to see your created items.
  • Edit Leaderboard View
  • On clicking Edit Layout button, you can enable Edit mode of Leaderboard.
  • In Edit mode, you can perform the following actions :
  • You can drag and drop the Leaderboard items as per your need.
  • You can modify the size of Leaderboard item as per your choice.
  • You can Edit the leaderboard name as well.
  • After performing all changes, you can either Save or Discard changes.
  • Leaderboard Manager
  • Leaderboard Manager has the following rights :
  • Can create/modify Leaderboards.
  • Can create/modify Leaderboards items.
  • Can Modify Leaderboard View.
  • In order to allow the user to view full Leaderboard Features, Go to Setting > Users & Companies > Users > Open User Form view
  • Under Application Access there would be a Selection field Leaderboard Rights.
  • Select Show Full Leaderboard Features to allow user.
