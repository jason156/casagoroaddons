Product Advance Barcode Labels

This module allows admin to generate product barcode labels of different size and
support multiple configuration for barcode label generation.

Features:

  • This module generates barcode label of different orientation: Landscape/portrait.
  • Provides feature of barcode label generation of different sizes: Small/Medium/Large.
  • Admin can add product images, description, and other product attributes on Barcode Label.
  • Add product expiry date and batch number.
  • Admin can create multiple label configuration that will be selected at the time of label download.
  • Admin can customize Barcode label layout by adding a suitable image, description, price, etc.
  • Admin can select Barcode field (HS Code/Barcode) that will be used to generate barcode labels.
  • Different price can be configured for different label layout using pricelist.
  • Admin can download the Demo pdf labels for label configurations.
  • Supports all barcode standards: EAN 13, UPC, Code 128, code 39 ... (+12 Barcodes)
  • Admin can generate Barcode labels in a single PDF for multiple products.
  • Mass Barcode Label Printing (pdf files).
